Github Copilot : où nous emmène-t-il ?

Github communique autour de son service Copilot, et dévoile quelques usages afin de le mettre en avant. Ainsi, Copilot permet d'envoyer des tweets. Très utile, surtout dans le contexte actuel, cet usage met surtout en lumière le fait que le service a aidé une développeuse à générer du Python sans maitriser le langage. Plus intéressant, sont aussi avancés des mise en pratique des capacité multilingues, allant de la traduction de cahines de caractères à la correction de règles CSS mal ortographiées. En interne, le responsable des relations développeurs de l'entreprise utilisé Copilot afin de créer un dictionnaire de liste de pays et leur code ISO à partir d'un commentaire et d'un fragment de code. Github met aussi en avant son extension expérimentale Copilot Labs, chargée de traduire du code dans un autre langage et expliquer le résultat étape par étape.
